Guía Completa de Vectores

1. ¿Qué es un Vector?

En física y matemáticas, un vector es una entidad que tiene tanto magnitud (o módulo) como dirección. Se diferencia de un escalar, que solo tiene magnitud (como la temperatura o la masa).

Ejemplos de vectores incluyen el desplazamiento, la velocidad, la aceleración y la fuerza.

Representación Gráfica

Un vector se representa gráficamente como una flecha. La longitud de la flecha representa su magnitud, y la punta de la flecha indica su dirección.

Representación Analítica (Componentes)

En un sistema de coordenadas (como el cartesiano x-y), un vector puede descomponerse en sus componentes. Estas son las "sombras" o proyecciones del vector sobre cada eje.

Para un vector $\vec{V}$ y un ángulo $\theta$ medido desde el eje x positivo:

Componente x: $V_x = |\vec{V}| \cos(\theta)$
Componente y: $V_y = |\vec{V}| \sin(\theta)$

Inversamente, si conoces las componentes, puedes encontrar la magnitud y la dirección:

Magnitud: $|\vec{V}| = \sqrt{V_x^2 + V_y^2}$ (Teorema de Pitágoras)
Dirección: $\theta = \arctan\left(\frac{V_y}{V_x}\right)$

2. Operaciones con Vectores

Suma de Vectores: $\vec{R} = \vec{A} + \vec{B}$

Método Gráfico (Cabeza con Cola): Se dibuja el primer vector $\vec{A}$. Luego, se dibuja el segundo vector $\vec{B}$ comenzando desde la punta (cabeza) de $\vec{A}$. El vector resultante $\vec{R}$ es la flecha que va desde el origen de $\vec{A}$ hasta la punta de $\vec{B}$.

Método Analítico (por Componentes): Es el método más preciso. Se suman las componentes correspondientes de cada vector.

$R_x = A_x + B_x$
$R_y = A_y + B_y$

Luego, se calcula la magnitud y dirección de $\vec{R}$ usando sus componentes $R_x$ y $R_y$.

Resta de Vectores: $\vec{R} = \vec{A} - \vec{B}$

Restar un vector es lo mismo que sumar su opuesto. El vector opuesto ($-\vec{B}$) tiene la misma magnitud que $\vec{B}$ pero apunta en la dirección contraria (180° de diferencia).

$\vec{R} = \vec{A} + (-\vec{B})$

Método Analítico: Se restan las componentes correspondientes.

$R_x = A_x - B_x$
$R_y = A_y - B_y$

Producto Escalar (Producto Punto): $c = \vec{A} \cdot \vec{B}$

El producto escalar de dos vectores da como resultado un escalar (un número, no un vector). Mide qué tan alineados están dos vectores. Una aplicación clave es el cálculo del Trabajo ($W = \vec{F} \cdot \vec{d}$).

Fórmula Geométrica (con el ángulo):

$\vec{A} \cdot \vec{B} = |\vec{A}| |\vec{B}| \cos(\theta)$
($\theta$ es el ángulo entre los dos vectores)

Fórmula Analítica (con componentes):

$\vec{A} \cdot \vec{B} = A_x B_x + A_y B_y$
