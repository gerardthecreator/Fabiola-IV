Suma Gráfica de Vectores

Enunciado

Se suman dos vectores $\vec{A}$ y $\vec{B}$ para obtener un vector resultante $\vec{C} = \vec{A} + \vec{B}$.

  • El vector $\vec{A}$ tiene una magnitud de 3.00 unidades y forma un ángulo de 60° con el eje x positivo (nordeste).
  • El vector $\vec{B}$ tiene una magnitud de 4.00 unidades y forma un ángulo de 40° con la dirección oeste (hacia el norte del oeste).

Usando el método gráfico y verificando con el método analítico, determine la magnitud de $\vec{C}$ y el ángulo que forma con la dirección oeste (eje -x).

Solución Detallada (Método Analítico)

1. Descomposición de Vectores

Primero, descomponemos cada vector en sus componentes $x$ e $y$. Para ello, necesitamos sus ángulos estándar (medidos desde el eje +x en sentido antihorario).

  • Vector A: El ángulo ya es estándar. $\theta_A = 60^\circ$.
  • Vector B: "40° con la dirección oeste" hacia el norte significa 40° desde el eje -x hacia el eje +y. El ángulo estándar es $\theta_B = 180^\circ - 40^\circ = 140^\circ$.
$A_x = |\vec{A}| \cos(\theta_A) = 3.00 \cdot \cos(60^\circ) = 1.50$
$A_y = |\vec{A}| \sin(\theta_A) = 3.00 \cdot \sin(60^\circ) = 2.60$

$B_x = |\vec{B}| \cos(\theta_B) = 4.00 \cdot \cos(140^\circ) = -3.06$
$B_y = |\vec{B}| \sin(\theta_B) = 4.00 \cdot \sin(140^\circ) = 2.57$

2. Suma de Componentes

Sumamos las componentes para encontrar las del vector resultante $\vec{C}$.

$C_x = A_x + B_x = 1.50 + (-3.06) = -1.56$
$C_y = A_y + B_y = 2.60 + 2.57 = 5.17$

3. Magnitud y Dirección de $\vec{C}$

Usamos el Teorema de Pitágoras para la magnitud y la arcotangente para la dirección.

Magnitud: $|\vec{C}| = \sqrt{C_x^2 + C_y^2} = \sqrt{(-1.56)^2 + (5.17)^2} = \sqrt{2.43 + 26.73} \approx 5.40 \text{ unidades}$

Ángulo estándar: $\theta_C = \arctan\left(\frac{C_y}{C_x}\right) = \arctan\left(\frac{5.17}{-1.56}\right) \approx 106.8^\circ$

El ángulo de $106.8^\circ$ es el medido desde el eje Este (+x). El problema pide el ángulo con el eje Oeste (-x). Como el vector está en el segundo cuadrante, este ángulo ($\alpha$) es:

$\alpha = 180^\circ - \theta_C = 180^\circ - 106.8^\circ = 73.2^\circ$

Respuesta Final

El vector resultante $\vec{C}$ tiene una magnitud de 5.40 unidades y forma un ángulo de 73.2° con la dirección Oeste (hacia el Norte).
