Encontrar un Vector Desconocido

Enunciado

Se tienen dos vectores en el plano cartesiano (xy):

  • Vector $\vec{A} = (5\i + 8\j)$
  • Vector $\vec{B}$ con componentes desconocidas $(B_x, B_y)$.

El vector resultante de la resta $\vec{R} = \vec{A} - \vec{B}$ es igual a $\vec{R} = (2\i + 11\j)$.

Determine las componentes del vector $\vec{B}$ ($B_x$ y $B_y$).

Solución Detallada

1. Reorganizar la Ecuación Vectorial

Partimos de la ecuación dada: $\vec{R} = \vec{A} - \vec{B}$. Nuestro objetivo es encontrar $\vec{B}$, por lo que debemos despejarlo algebraicamente.

$\vec{R} = \vec{A} - \vec{B}$
$\vec{R} + \vec{B} = \vec{A}$
$\vec{B} = \vec{A} - \vec{R}$

Esto nos dice que para encontrar el vector $\vec{B}$, debemos restar el vector $\vec{R}$ del vector $\vec{A}$.

2. Restar Componente a Componente

Ahora aplicamos la resta de vectores usando sus componentes. La componente $x$ de $\vec{B}$ es la resta de las componentes $x$ de $\vec{A}$ y $\vec{R}$, y lo mismo para la componente $y$.

Componente x: $B_x = A_x - R_x = 5 - 2 = 3$

Componente y: $B_y = A_y - R_y = 8 - 11 = -3$

3. Escribir el Vector Resultante

Con las componentes calculadas, podemos escribir el vector $\vec{B}$ completo.

$\vec{B} = (3\i - 3\j)$

Respuesta Final

Las componentes del vector $\vec{B}$ son:
$B_x = 3$
$B_y = -3$
