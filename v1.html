Suma Gráfica de Vectores

Enunciado

Sean $\vec{A}$ y $\vec{B}$ los desplazamientos de 3.00 km hacia el Este y 4.00 km hacia el Norte, respectivamente. Sea $\vec{C} = \vec{A} + \vec{B}$.

Usando el método gráfico (cabeza con cola) y una escala de 1 cm = 1 km, determine el módulo y la dirección del vector resultante $\vec{C}$.

Construcción Gráfica

Presiona "Siguiente" para construir el diagrama.

Solución Detallada

1. Interpretación Gráfica (Método del Dibujo)

Siguiendo las instrucciones, dibujamos los vectores a escala. El vector $\vec{A}$ se dibuja de 3 cm hacia el Este. Desde la punta de $\vec{A}$, se dibuja el vector $\vec{B}$ de 4 cm hacia el Norte. El vector resultante $\vec{C}$ va desde el origen de $\vec{A}$ hasta la punta de $\vec{B}$.

Módulo de C: Al medir la longitud del vector $\vec{C}$ en el dibujo con una regla, obtendríamos 5.00 cm. Usando la escala, esto equivale a 5.00 km.

Dirección de C: Al usar un transportador para medir el ángulo $\theta$ que forma $\vec{C}$ con el eje Este (vector $\vec{A}$), obtendríamos aproximadamente 53°.

2. Verificación Analítica (Cálculo)

Podemos confirmar los resultados del método gráfico usando el Teorema de Pitágoras para el módulo y trigonometría para la dirección, ya que los vectores forman un triángulo rectángulo.

Módulo de C: $|\vec{C}| = \sqrt{|\vec{A}|^2 + |\vec{B}|^2} = \sqrt{(3.00)^2 + (4.00)^2} = \sqrt{9.00 + 16.0} = \sqrt{25.0} = 5.00 \text{ km}$

Dirección de C: $\theta = \arctan\left(\frac{|\vec{B}|}{|\vec{A}|}\right) = \arctan\left(\frac{4.00}{3.00}\right) \approx 53.1^\circ$

Los cálculos confirman que el método gráfico es una forma visualmente precisa de sumar vectores.

Respuesta Final

El vector resultante $\vec{C}$ tiene un módulo de 5.00 km y una dirección de 53.1° al Norte del Este.
